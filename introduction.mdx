## Introduction

This tutorial will have you deploying a Node.js app to GL Cloud in minutes.

The tutorial assumes that you have a [GL Cloud account](https://glcloud.app/login), and that you have [Node.js](https://nodejs.org/en/download/) and [npm](https://www.npmjs.com/package/npm) installed locally.


## Required Files

> index.js or index.ts (Example of main file).
> package.json (dependency file).
> squarecloud.app (file that contains the Square Cloud configuration).
