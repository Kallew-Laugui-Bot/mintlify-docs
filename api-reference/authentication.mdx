First you need to get an API key from the [Developer Dashboard](https://glcloud.app/dashboard/account). You can use this API key to authenticate to the GL Cloud API.

```bash
'Authorization': '<api_key>'
```

## API Usage
You can use the API key to authenticate to the GL Cloud API. You can use the API key in the `Authorization` header.
Remembering that the use of the API is subject to the [Terms of Service](https://glcloud.app/legal) and the [Acceptable Use Policy](https://glcloud.app/legal/policy).
