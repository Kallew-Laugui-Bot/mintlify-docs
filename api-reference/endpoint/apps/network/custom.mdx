**Is necessary to have at least the [senior plan](https://glcloud.app/plans) to use this route.**

<ParamField header="Authorization" type="string" placeholder="API KEY" required>
  The API key for your account. You can find this in your [account settings](https://glcloud.app/dashboard/account).
</ParamField>

<ParamField path="app_id" type="string" placeholder="Application ID" required>
  The ID of the application. You can find this in the URL of your application's dashboard.
</ParamField>

<ParamField path="custom_domain" type="string" placeholder="example.com" required>
  The custom domain you want to use.
</ParamField>
